# Asteroids Rust

A simple Asteroids game implementation using Bevy game engine written in Rust.

<div align="center">
  <img src="Example.gif" alt="Gameplay Example">
</div>

## Controls

- Move the player ship with WASD
- Press Space to toggle wireframes

## Building and Running

```bash
cargo run --release
```

## Dependencies

- Bevy 0.16.0
